# Description
Простое iOS приложение для конвертации валют

# Stack
UIKit - Выбрал его для того чтобы закрепить тему вёрстки интерфейсов с помощью кода
MVVM - Простая и популярная архитектура в которой сможет разобраться любой разработчик, легко позволяет писать юнит тесты которые нужны были в этом проекте

# Обработанные нестандартные ситуации
- Не получилось загрузить курс - показываем захардкоженный курс по валюте usdc_ars
- При вводе 0 в противоположном поле ничего не происходит
